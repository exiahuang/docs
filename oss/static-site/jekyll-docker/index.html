    jekyll-docker

    To install Jekyll in a Docker environment using a Gemfile, follow these steps:

    1. Create a Gemfile¶

    Inside your project directory, create a file named Gemfile with the following content:

    source "https://rubygems.org"

gem "jekyll"

    2. Create a Dockerfile¶

    Now, create a Dockerfile to define the Jekyll environment:

    FROM ruby:latest

WORKDIR /usr/src/app

COPY Gemfile Gemfile.lock ./

RUN bundle install

COPY . .

EXPOSE 4000

CMD ["bundle", "exec", "jekyll", "serve", "--host", "0.0.0.0"]

    3. Build and Run the Container¶

    Run the following commands to build and start the container:

    docker build -t jekyll-site .
docker run -p 4000:4000 -v $(pwd):/usr/src/app jekyll-site

    This will install Jekyll inside the container and serve the site at http://localhost:4000.
