    init-springboot

    在 Maven 中，你可以使用 mvn archetype:generate 命令来创建一个 Spring Boot 项目。以下是详细步骤：

    1. 使用 Maven 创建 Spring Boot 项目¶

    在终端或命令行中运行以下命令： 

    mvn archetype:generate \
    -DgroupId=com.example \
    -DartifactId=springboot-demo \
    -DarchetypeArtifactId=maven-archetype-quickstart \
    -DinteractiveMode=false
    这将会创建一个基本的 Maven 项目。

    2. 添加 Spring Boot 依赖¶

    进入项目目录： 

    cd springboot-demo
    打开 pom.xml，修改 <dependencies>，添加 Spring Boot 依赖： 
    <dependencies>
    <!-- Spring Boot Starter -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter</artifactId>
        <version>3.2.0</version>
    </dependency>

    <!-- Spring Boot Web -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>

    <!-- Spring Boot Test -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-test</artifactId>
        <scope>test</scope>
    </dependency>
</dependencies>

    在 <build> 部分，添加 Spring Boot Maven 插件： 

    <build>
    <plugins>
        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
        </plugin>
    </plugins>
</build>

    3. 创建 Spring Boot 主类¶

    在 src/main/java/com/example 目录下创建 SpringbootDemoApplication.java： 

    package com.example;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class SpringbootDemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(SpringbootDemoApplication.class, args);
    }
}

    4. 运行 Spring Boot 应用¶

    在终端中执行： 

    mvn spring-boot:run
    如果看到： 
    Tomcat started on port 8080
    说明 Spring Boot 项目启动成功 🎉！

    5. 测试¶

    创建一个简单的 Controller，在 com.example 目录下创建 HelloController.java： 

    package com.example;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
@RequestMapping("/api")
public class HelloController {
    @GetMapping("/hello")
    public String sayHello() {
        return "Hello, Spring Boot!";
    }
}
    然后，在浏览器访问： 
    http://localhost:8080/api/hello
    应该能看到： 
    Hello, Spring Boot!

    6. 其他方式（更快捷）¶

    你也可以使用 Spring Boot 官方的 spring-boot-starter-parent 快速创建： 

    mvn archetype:generate \
    -DgroupId=com.example \
    -DartifactId=springboot-demo \
    -DarchetypeArtifactId=maven-archetype-quickstart \
    -DinteractiveMode=false
    然后修改 pom.xml： 
    <parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>3.2.0</version>
</parent>
    最后运行： 
    mvn spring-boot:run

    这样，你就成功创建了一个 Spring Boot 项目 🎉！
